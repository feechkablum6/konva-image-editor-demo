# Konva Image Editor Demo (Vue 3)

Демо версия редактора изображений: блоки на канвасе, редактирование и экспорт.

## Стек

- Vue 3
- Vite
- Konva
- vue-konva

## Что реализовано в демо

- Загрузка изображений (`image/*`, множественный выбор)
- Кроппер при загрузке и повторный кроп уже добавленного изображения
- Трансформации блока: перетаскивание, resize, rotate
- Отражение изображения по X/Y
- Цветокоррекция: brightness, contrast, saturation, hue
- Рисование/ретушь (базовый MVP): кисть и ластик
- Добавление и редактирование текстового блока
- Работа с цветом: color picker + EyeDropper API (если поддерживается браузером)
- История изменений: undo/redo на 5 шагов
- Навигация по редактору: zoom колесом, pan mode, reset view, center selected
- Экспорт изображения: PNG/JPEG/WebP + quality + pixel ratio + width/height
- Импорт/экспорт состояния редактора в JSON (для демонстрации работоспособности)

## Запуск

```bash
npm install
npm run dev
```

Открыть ссылку из Vite (обычно `http://localhost:5173`).

## Быстрый сценарий проверки

1. Загрузить 1-2 изображения через `Upload image(s)`.
2. В кроппере показать `Centered square`, `Rotation`, `Apply crop`.
3. Выделить блок, сделать `Flip X`/`Flip Y`, подвигать с resize/rotate.
4. Изменить `Brightness/Contrast/Saturation/Hue`.
5. Включить `Brush`, сделать пару штрихов, затем `Eraser`.
6. Добавить `Add text`, изменить текст/цвет/размер.
7. Нажать `Undo`/`Redo` (до 5 шагов).
8. Показать экспорт в PNG и JPEG/WebP с разным quality.

## Ограничения MVP

- Продвинутые clone/heal инструменты не реализованы (только базовая ретушь кистью/ластиком).
- EyeDropper зависит от поддержки браузера.
